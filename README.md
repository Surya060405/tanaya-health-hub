# Tanaya Health Hub

## About the Project
Tanaya Health Hub is an AI-powered companion for gynecological health, guidance, and care. From periods to pregnancy, hormones to fertility, TANAYA provides personalized insights, expert support, and safe digital care for women at every stage of life.

## Features
- **AI-Powered Health Insights**: Personalized guidance for your unique health journey.
- **Cycle Tracking**: Monitor your menstrual cycle, ovulation, and symptoms.
- **Expert Support**: Access to reliable medical information and professional advice.
- **Community**: Connect with a supportive community of women.

## Tech Stack
This project is built with modern web technologies:
- **Vite**: Next Generation Frontend Tooling
- **React**: A JavaScript library for building user interfaces
- **TypeScript**: Typed JavaScript at Any Scale
- **Tailwind CSS**: A utility-first CSS framework
- **Shadcn UI**: Reusable components built with Radix UI and Tailwind CSS

## Getting Started

### Prerequisites
- Node.js & npm installed

### Installation
1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd tanaya-health-hub
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Building for Production
To build the app for production, run:
```sh
npm run build
```
